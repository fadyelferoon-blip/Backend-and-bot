## 🚀 النشر على Railway:

### خطوة خطوة:

1. افتح https://railway.app
2. اضغط "New Project"
3. اختر "Deploy from Local" أو "Empty Project"
4. **ارفع:** `index.html` فقط
5. **اسم المشروع:** FER3OON-Bot-MXN
6. Railway يكشف تلقائياً: **Static Site**
7. اضغط "Deploy"
8. انتظر 1-2 دقيقة
